# 🌦️ Weather App

A modern and responsive Weather App built using **HTML, CSS, and JavaScript**. It fetches real-time weather data using a Weather API and displays current weather information for any city worldwide.

## 🚀 Features

- 🔍 Search weather by city name
- 🌡️ Real-time temperature
- ☁️ Weather condition and description
- 💨 Wind speed
- 💧 Humidity
- 🌍 Worldwide city support
- ⚡ Fast and user-friendly interface

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Weather API
- Fetch API
